

1️⃣ Crear el entorno virtual:

python3.11 -m venv p1_env
2️⃣ Activar el entorno virtual:

source p1_env/bin/activate  # En Linux/macOS  
p1_env\Scripts\activate     # En Windows
3️⃣ Instalar las dependencias dentro del entorno virtual:

pip3 install -r requirements.txt

2️⃣ Asegurar que venv está instalado:
sudo apt update
sudo apt install python3.11-venv

(si falla)
Para Debian/Ubuntu:
sudo apt update
sudo apt install python3.11-dev libpq-dev gcc
